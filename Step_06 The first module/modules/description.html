In the markup, complex structures can be marked as composite and thus form a module that can consist of markup (HTML), JavaScript and CSS. A corresponding JavaScript object (model) can be implemented for the markup. Model and markup are linked by the renderer and the object binding.

With the link between model and markup, the object binding can forward data and events between the two. Synchronization, validation and events can be defined by declaration and can then be used in the model.
