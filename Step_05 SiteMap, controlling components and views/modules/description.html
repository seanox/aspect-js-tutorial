The SiteMap in Seanox aspect-js is an optional helper for controlling the view and webflows. With it the complete webflow with all views (Faces) and partial views (Factes) is mapped as virtual paths. This allows the SiteMap to monitor and control which elements must be shown for a path. In addition, the integration of permission concepts is possible here.
